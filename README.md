# PIXVOY - London Photography Walk

A curated 1-hour photography journey through the heart of London, designed to help you master mobile photography while exploring the city's most iconic locations.

## The Experience

PIXVOY offers intimate, small-group photography walks limited to 8-12 participants. Perfect for iPhone users looking to maximize their gear, beginner to intermediate photographers, content creators, and anyone wanting a creative perspective on London.

## The Route

Five carefully selected stops showcasing London's unique visual rhythm:

| Stop | Location | Focus |
|------|----------|-------|
| 01 | **Tate Modern** | Industrial textures and iconic brick Switch House architecture |
| 02 | **Millennium Bridge** | Leading lines and the silver walkway towards the Thames |
| 03 | **St Paul's Cathedral** | Mirror reflections of the grand cathedral dome |
| 04 | **Bloomberg Arcade** | Modern rhythmic geometry and bronze-finned architecture |
| 05 | **Bank Junction** | Classical Royal Exchange meeting modern glass giants |

## Details

- **Duration:** 1 hour
- **Start Time:** 8 AM
- **Meeting Point:** Tate Modern main entrance
- **Group Size:** 8-12 participants
- **What to Bring:** Fully charged iPhone, comfortable walking shoes, weather-appropriate clothing

## Development

### Prerequisites
- Node.js 20+

### Run Locally

```bash
npm install
npm run dev
```

The app will be available at `http://localhost:3000`

### Build

```bash
npm run build
```

### Deploy

The site automatically deploys to GitHub Pages on push to `main`.
